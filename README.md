# 📚 Personal Project Showcase App

A modern, responsive React application for managing and showcasing personal projects with search functionality.

## ✨ Features

- **Add New Projects** - Create projects with title and description
- **Real-time Search** - Filter projects by title or description instantly
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Modern UI** - Clean, intuitive interface with smooth animations
- **No Backend Required** - Client-side storage for quick prototyping
